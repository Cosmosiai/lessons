---Что такое git?
Git - это система, которая помогает в разработке программ. С помощью git можно отправлять код в общее место, где команда программистов сможет совместно работать над проектом.
---Для чего нужны системы контроля версий?
При работе может появиться необходимость в возращении к старым версиям программ. Также может появиться необходимость в отслеживании изменений и кем они были добавлены. Для того и нужна эта система.
---В чем разница между удаленным и локальным репозиториями?
Удалённый репозиторий не есть твой компьютер.
---Что делает команда commit?
Commit берёт добавленный (с помощью add) код программы и сохраняет его как слепок во внутренней памяти. Затем он указывает что-то что мы закоммитили и указывает это как последнюю редакцию кода.
---Что делает команда push?
Эта команда отправляет код в удалённый репозиторий (в гитхаб)
---Для чего нужны ветки
Чтобы когда команда программистов изменяли код, код не путался. Каждый будет работать в своей ветке.
